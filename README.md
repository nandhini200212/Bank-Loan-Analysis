Bank Loan Analysis

This project focuses on analyzing bank loan data to evaluate loan performance, customer behavior, repayment trends, and portfolio quality. Using Power BI, Tableau, SQL, and Excel, interactive dashboards and reports were developed to provide insights that support lending decisions and risk management.

Tools Used
Power BI
Tableau
SQL
Microsoft Excel

Key Features
Loan Application Analysis
Funded Amount and Loan Performance Tracking
Customer Demographics Analysis
Repayment and Default Trend Analysis
KPI Dashboard Development
Data Cleaning and Transformation
Interactive Reports and Visualizations

Key KPIs
Total Loan Applications
Total Funded Amount
Total Amount Received
Average Interest Rate
Average Debt-to-Income Ratio
Good Loan vs Bad Loan Analysis
Outcome

Provided actionable insights into loan portfolio performance, customer lending patterns, and repayment behavior, enabling data-driven decision-making and effective risk assessment.
