# Bank Loan Analysis

## Project Overview

This project focuses on analyzing bank loan data to evaluate loan performance, customer behavior, repayment trends, and portfolio quality. The analysis was performed using Power BI, Tableau, SQL, and Excel to generate actionable insights and support data-driven lending decisions.

## Tools Used

* Power BI
* Tableau
* SQL
* Microsoft Excel

## Objectives

* Analyze loan applications and funded amounts.
* Evaluate repayment performance and portfolio quality.
* Identify good loans and bad loans.
* Understand customer demographics and borrowing patterns.
* Support lending and risk management decisions through data insights.

## Key Insights

* Loan Application Analysis
* Funded Amount & Loan Disbursement Analysis
* Good Loan vs Bad Loan Analysis
* Customer Demographics Analysis
* Loan Repayment Performance
* Interest Rate & Debt-to-Income Analysis
* Regional Loan Distribution Analysis
* Portfolio Quality & Risk Assessment

## Key KPIs

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income Ratio (DTI)
* Good Loan Percentage
* Bad Loan Percentage

## Process

1. Collected and reviewed bank loan data.
2. Cleaned and transformed data using Excel and SQL.
3. Performed data modeling and established table relationships.
4. Built interactive dashboards in Power BI and Tableau.
5. Analyzed loan performance and generated business insights.

## Outcome

The project provided valuable insights into loan portfolio performance, customer lending behavior, and repayment trends, enabling informed decision-making and effective risk assessment.
